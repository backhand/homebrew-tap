# backhand/homebrew-tap

Homebrew tap for [backhand](https://github.com/backhand) command-line tools.

## kc — kubernetes commander

```sh
brew tap backhand/tap
brew install kc
# …or in one line:
brew install backhand/tap/kc
```

Casks here are generated automatically on each release by
[GoReleaser](https://goreleaser.com); see the source repo
([backhand/kc](https://github.com/backhand/kc)) for the tool itself.
